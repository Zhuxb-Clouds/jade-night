# 叙事设计

## 行香子·春夜宴
翠幕烟浮，红烛影流。满后园、莺语难收。香糕细点，翠袖轻勾。正宴芳华，分春色，在心头。
老亲垂眸，玉盏清幽。递温润、且试温柔。钗环并凑，争看此瓯。愿光如旧，人如旧，岁如流。

## 故事大概

### 正面

每年暖春时节夜里，顾府后院总会点灯开宴，顾府的女子们莺莺燕燕一起聊天、唱歌、品尝点心。席间，顾府老夫人会将自己的玉盏给参加宴席的一个晚辈，表达她的喜爱之情。

# 结构设计

## 一、 核心分值定义

游戏中核心得分来源是**配对分**。

* **配对分（Points）：** 指一组"食器"与"点心"的匹配得分。

**【配对分计算】**
* **基础分** = 匹配的维度数（颜色、形状、材质，0-3分）
* **完美奖励** = 3个维度全匹配时 +1分
* **最终配对分** = 基础分 + 完美奖励
* **可能的分值：** 0, 1, 2, 4分（没有3分！）

* **视觉规则：** 
    * 卡牌设计（点心牌）为透明材质，牌面上设有**三列**数据位，分别对应上述三个维度。
    * **点心牌**在对应维度若有特征，显示为**实心圆点（点）**。
    * **食器牌**在对应维度若有特征，显示为**空心圆环（圈）**。
    * 当点心放置于食器之上时，若"点"恰好位于"圈"内（形成**同心圆**），则该维度匹配成功。

---

## 二、 游戏流程

### 1. 区域划分

1. 公共区： 展示可取用的一级食器与点心。
2. 等待区： 玩家尚未结算的食器与点心。上限为 **5**。
3. 个人区： 玩家已完成“独食”结算的点心，上限为 **6**。
4. 奉献区： 玩家献给老太君的点心组合。

### 公共区详解

公共区牌列设置为 **5个槽位**。
每个槽位里有一个点心，拿走将立刻补充。
公共区域有三个独立的食器牌堆：
- **L1 牌堆**：玩家可消耗行动点从此抽取 Level 1 食器
- **L2 牌堆**：通过奉献 L1 食器获得
- **L3 牌堆**：通过奉献 L2 食器获得


### SetUp

*   所有玩家初始获得 1 个 Level 1 食器。
*   L1、L2、L3 食器各自独立成堆，分别放置。


### 2. 玩家行动

每回合，玩家获得基础行动点（AP）**3点**。

**茶歇机制 (Tea Break)：**
* 个人区每完成 **2盘** 点心结算，获得一枚 **【茶券】（Tea Token）**。
* **效果：** 玩家可以在自己的回合，消耗一枚【茶券】来获得 **+1 AP**。

玩家可以选择以下动作消耗行动点：

**A. 拿取食器/点心 (1 AP)**

* 从公共区拿取一张**食器**/**点心**放入等待区。
* 从公共区拿取一张点心，放置在等待区中有空位的食器上。
* **注意：** 每个食器仅限承载一张点心，不可叠加。
* 公共区立即从牌堆补充。

**B. 品鉴（独食得分） (1 AP)**

* 将等待区的一组“食器+点心”移入个人区。
* 此后该组合不可变动，其配对分计入最终得分。
* **限制：** 每回合只能品鉴 **1次**。

**C. 奉献（升级机制） (1 AP)**

* **过程：** 将等待区的组合移至奉献区，奉献点心给老太君时，玩家会获得1枚茶券。
* **含义：** 玩家展示摆盘技巧，老太君赏赐更好的器皿。 
* **限制：** 奉献的点心的配对分至少要>=2点。
* **特殊奖励：** 若奉献的点心配对分>=4点（完美配对），则额外获得一枚茶券，并**获得玉盏**（若玉盏已被他人持有则转移）。
* **奖励：** 玩家获得老太君回赠的一个**更高等级**的随机空食器（从奖励牌堆抽取 L2 或 L3）。
    *   L1 -> 获得 L2
    *   L2 -> 获得 L3
    *   L3 -> 获得两个茶券

**【奉献区得分】**
* 奉献区的点心**不计配对分**，只按食器等级计分：
    * L1 粗瓷盘：1分
    * L2 细釉盘：2分
    * L3 珍宝盘：3分
* 持有玉盏者，奉献区总分**翻倍**。

**D. 进入调整模式 (1 AP)**

* 消耗 **1 AP** 进入调整模式。
* **调整模式中：**
  - 等待区的点心会从盘子上浮起，显示为可拖动状态。
  - 你可以任意拖拽点心到任意空盘子上。
  - 调整模式期间，不能执行其他动作（拿取、品鉴、奉献等）。
* **退出调整模式：** 点击"完成调整"按钮，免费退出调整模式。
* **限制：** 每回合只能进入调整模式 **1次**。

**E. 赠尝 (Gifting) (1 AP)**

* 消耗 **1 AP**，将公共区的一张点心放置在**对手**等待区的空闲食器上。
* **限制：** 放置的点心必须至少能得 **1分**（不能送完全不匹配的组合）。
* **对手选择：** 被赠尝的玩家可以选择：
  - **接受：** 哪怕不喜欢这个点心，也含泪收下。盘子被占用，但获得小分。
  - **拒绝：** 消耗 **1枚茶券** 拒绝这份"好意"。点心被弃置。
    - *玉盏持有者拒绝需消耗 **2枚茶券**。*
    - *如果茶券不足，则无法拒绝，只能接受。*
* **消耗战博弈：**
  - 发起方代价：消耗 1 AP。如果对方拒绝，发起方空过了一次行动。
  - 接收方代价：接受则盘子被占；拒绝则消耗茶券。
  - 双方都有损失，迫使玩家深思熟虑。
* **策略意义：**
  - 当对手茶券富裕时，赠尝可能只是浪费自己的 AP。
  - 当对手茶券紧张时，赠尝成为有效的骚扰手段。
  - 对手需要权衡：是保住空盘子的灵活性，还是保住稀缺的茶券？
  - **针对玉盏持有者的赠尝尤为有效——拒绝代价翻倍！**

## 三、 老太君判定逻辑


玩家将制作好的点心献给老太君，虽不能品尝其美味（不计配对分），但能换取更高等级的食器，并积累“资历”。

### 1. 奉献规则
*   **动作：** 消耗 **1 AP**，将等待区的一组“食器+点心”移至奉献区。
*   **门槛（呈样）：** 被奉献的组合必须至少包含 **2点** 有效配对分。
*   **代价：** 奉献区的所有卡牌**不计算**“颜色/形状/材质”的配对得分。

### 2. 奉献奖励
当你进行奉献时，根据你献出的食器等级，老太君会从对应的独立牌堆中回赠礼品：
*   献 **Level 1** 食器 $\rightarrow$ 从 **L2 牌堆** 抽取 Level 2 食器（放入等待区）
*   献 **Level 2** 食器 $\rightarrow$ 从 **L3 牌堆** 抽取 Level 3 食器（放入等待区）
*   献 **Level 3** 食器 $\rightarrow$ 获得 **2 枚茶券**

### 4. 流动的玉盏

玉盏是一个象征老太君青睐的流动奖励，玩家通过奉献完美配对的点心来获得。

**获取与转移规则：**
*   当玩家**奉献**的点心组合配对分 **= 4分**（完美配对）时，该玩家获得**玉盏**。
*   如果玉盏已被其他玩家持有，则玉盏会**转移**到当前奉献者手中。
*   *叙事：只有手艺最精湛、奉献最诚挚的晚辈，才能赢得老太君的玉盏。*

**持有玉盏的特权与代价：**
*   **回合开始奖励：** 玉盏持有者在每回合开始时，获得 **+1 茶券**。
    *   *叙事：老太君格外关照持有玉盏的晚辈，赐予额外的好茶。*
*   **拒绝赠尝代价翻倍：** 玉盏持有者拒绝**赠尝**时，需消耗 **2枚茶券**（而非1枚）。
    *   *叙事：持有玉盏者受人瞩目，更难拒绝他人的"好意"。*

**最终结算加成：**
*   游戏结束时，玉盏持有者的**奉献区分数翻倍**。

**策略意义：**
*   玉盏不是一次性获取，而是动态流动的——你需要持续奉献高质量点心才能保住它。
*   持有玉盏虽有茶券收益，但也成为赠尝的优先目标（拒绝代价高昂）。
*   最终结算的翻倍效果非常可观，值得为之争夺。

---
## 四、 最终结算 (Final Scoring)

当满足以下条件时，游戏在进行到下一轮（公平轮）之后结束：
* **所有 L2 等级** 的食器都已被分发完毕。

### 1. 分值构成公式
每位玩家的最终得分 ($S$) 由以下各项加总构成：

$$S = \sum P_{ind} + C_{off} + T_{remain} - C_{wait} \times 2$$

如果持有玉盏，则 $C_{off} \times 2$

### 2. 各项变量详解

| 变量符号         | 名称             | 定义与计算方式                                                                       |
| :--------------- | :--------------- | :----------------------------------------------------------------------------------- |
| **$P_{ind}$**    | **个人区配对分** | 结算至个人区域的点心的配对分（含完美奖励）。                                         |
| **$C_{off}$**    | **奉献区得分**   | 按食器等级计分：L1=1分, L2=2分, L3=3分。持有玉盏则翻倍。                             |
| **$T_{remain}$** | **剩余茶券**     | 游戏结束时，玩家剩余的茶券数量。                                                     |
| **$C_{wait}$**   | **滞留惩罚**     | 游戏结束时，留在玩家等待区（未结算）的上面存在点心的**盘子数量**。每盘扣除 **2分**。 |

### 3. 胜负判定
* **总分最高者获胜。**
* **平局判定：** 若总分相同，则奉献区盘数（$C_{off}$）较多的玩家获胜；若仍相同，则个人区内盘子较少的玩家获胜。

# 卡牌设计

### 一、 核心公式：维度矩阵

为了让配对既有挑战又有运气，我们采用 **3 x 3 x 2** 的非对称矩阵设计。

*   **视觉（颜色 C）：** 3种 —— 桃红、柳绿、鹅黄
*   **哲学（形状 S）：** 3种 —— 圆满（圆）、方正（方）、花韵（花）
*   **体感（材质 M）：** 2种 —— 暖（温润/陶木）、冷（清冽/琉璃）
    *   *设计意图：材质只有2种，意味着“奉献”的门槛较低（50%概率匹配），保证了游戏推进的速度（N值增长），符合“后盘必出”的节奏。*

---

### 二、 点心牌设计 (Dessert Deck)

点心是消耗品，也是分数的载体。**点心没有等级区分**，所有点心都是单属性的基础形态。

#### 1. 数量公式
$$Total = 3(C) \times 3(S) \times 2(M) \times 3(R) = \mathbf{54张}$$

*   **颜色 (C)：** 3种 —— 桃红、柳绿、鹅黄
*   **形状 (S)：** 3种 —— 圆满（圆）、方正（方）、花韵（花）
*   **材质 (M)：** 2种 —— 暖、冷
*   **复本 (R)：** 每种组合有 3 张

#### 2. 分布详情

**A. 唯一组合 (18种)**
每种特定组合（如：桃红-圆-暖）在牌堆里有 **3张复本**，提供稳定的配对基础。

**B. 统计总览**
*   **颜色分布：** 红(18)、绿(18)、黄(18) —— 完美平衡。
*   **形状分布：** 圆(18)、方(18)、花(18) —— 完美平衡。
*   **材质分布：** 暖食(27)、冷食(27) —— 完美平衡。

#### 3. 牌面叙事包装（示例）

*   **桃红-圆-暖**：玫瑰赤豆糕
*   **柳绿-方-冷**：翡翠薄荷冻
*   **鹅黄-花-暖**：金桂海棠酥

---

### 三、 食器牌设计 (Plate Deck)

食器是得分倍率器，采用 **金字塔稀有度** 设计。
食器同样拥有 C/S/M 三个属性，但属性代表的是 **"兼容性"**。
食器等级越高，兼容的属性越广，越容易达成配对。

**总数量：36张 + 玉盏（特殊标识物）**

**注：新配对分规则下，只要三个维度都能匹配，无论L1/L2/L3都能获得完美配对分4分。食器等级影响的是匹配的容易程度，而非分数上限。**

#### Level 1：粗瓷盘 (Common)
*   **数量**：18张
*   **特征**：**单属性严格限制**。每个维度只接受1种属性。
*   **公式**：每个盘子只有一个固定的颜色、形状和材质。
    *   例如：[红 / 圆 / 暖]
*   **设计意图**：难以匹配完美配对，主要用于奉献升级。
*   **奉献得分**：1分

#### Level 2：细釉盘 (Rare)
*   **数量**：12张 (老太君回赠的主要池子)
*   **特征**：**双维度宽容**。某一维度接受2种属性。
*   **设计逻辑**：分为三类，每类4张：
    *   **双色盘**：颜色维度兼容2种颜色
    *   **双形盘**：形状维度兼容2种形状
    *   **温润盘**：材质维度兼容冷热
*   **奉献得分**：2分

#### Level 3：珍宝盘 (Epic)
*   **数量**：6张 (稀有掉落)
*   **特征**：**单维度全通（万能）**。某一维度兼容所有属性。
*   **牌面示例**：
    *   **流光盘**（3张）：颜色维度全通，兼容所有颜色
    *   **百花盘**（3张）：形状维度全通，兼容所有形状
*   **奉献得分**：3分

#### 特殊：玉盏 (Legendary)
*   **数量**：全场只存在一个。
*   玉盏只作为标识物，不占用等待/个人/奉献区域。持有玉盏者奉献区分数翻倍计算。
