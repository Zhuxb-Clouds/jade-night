# 叙事设计

## 行香子·春夜宴
翠幕烟浮，红烛影流。满后园、莺语难收。香糕细点，翠袖轻勾。正宴芳华，分春色，在心头。
老亲垂眸，玉盏清幽。递温润、且试温柔。钗环并凑，争看此瓯。愿光如旧，人如旧，岁如流。

## 故事大概

### 正面

每年暖春时节夜里，顾府后院总会点灯开宴，顾府的女子们莺莺燕燕一起聊天、唱歌、品尝点心。席间，顾府老夫人会将自己的玉盏给参加宴席的一个晚辈，表达她的喜爱之情。

# 结构设计

## 一、 核心分值定义

游戏中核心得分来源是**配对分**。

* **配对分（Points）：** 指一组"食器"与"点心"的匹配得分。
* 盘子上的每一个"圈"若被点心的"点"填满，即产生 **1点** 配对分。
  * 例如：双色盘[红+绿]有2个颜色圈，若配上双色点心[红+绿]，则颜色维度得 **2分**。
  * 例如：L1单色盘[红]配单色点心[红]，颜色维度得 **1分**。
* **视觉规则：** 
    * 卡牌设计（点心牌）为透明材质，牌面上设有**三列**数据位，分别对应上述三个维度。
    * **点心牌**在对应维度若有特征，显示为**实心圆点（点）**。
    * **食器牌**在对应维度若有特征，显示为**空心圆环（圈）**。
    * 当点心放置于食器之上时，若"点"恰好位于"圈"内（形成**同心圆**），则该圈被填满，计入1分。
* 食器分为三个等级，越高级的食器拥有的"圈"越多，配对分上限也越高。

---

## 二、 游戏流程

### 1. 区域划分

1. 公共区： 展示可取用的一级食器与点心。
2. 等待区： 玩家尚未结算的食器与点心。上限为 **5**。
3. 个人区： 玩家已完成“独食”结算的点心，上限为 **5**。
4. 奉献区： 玩家献给老太君的点心组合。

### 公共区详解

公共区牌列设置为 **9个槽位**，按照九宫格排列。
每个槽位里有一个点心和一个食器，点心在食器上，想要拿走食器必须先拿走上面的点心。
如果同排/同列的盘子上都没有点心，则刷新这一排的点心和盘子。
**惜食**： 拿走同排/同列里最后一个点心的人，获取一个茶券，如果同时同排/同列都没有，则获取两个。
当L1盘子被消耗完毕，则拿去L2盘子作为基底。


### SetUp

*   所有玩家初始获得 1 个 Level 1 食器。
*   公共牌堆仅包含 Level 1 食器。
*   Level 2 与 Level 3 食器组成“奖励牌堆”。


### 2. 玩家行动

每回合，玩家获得基础行动点（AP）**3点**。

**茶歇机制 (Tea Break)：**
* 个人区每完成 **2盘** 点心结算，获得一枚 **【茶券】（Tea Token）**。
* **效果：** 玩家可以在自己的回合，消耗一枚【茶券】来获得 **+1 AP**。

玩家可以选择以下动作消耗行动点：

**A. 拿取食器/点心 (1 AP)**

* 从公共区拿取一张**食器**/**点心**放入等待区。
* 从公共区拿取一张点心，放置在等待区中有空位的食器上。
* **注意：** 每个食器仅限承载一张点心，不可叠加。
* 公共区立即从牌堆补充。

**B. 品鉴（独食得分） (1 AP)**

* 将等待区的一组“食器+点心”移入个人区。
* 此后该组合不可变动，其配对分计入最终得分。
* **限制：** 每回合只能品鉴 **1次**。

**C. 奉献（升级机制） (1 AP)**

* **过程：** 将等待区的组合移至奉献区。
* **含义：** 玩家展示摆盘技巧，老太君赏赐更好的器皿。 
* **限制：** 奉献的点心的配对分至少要>=2点。
* **特殊奖励：** 若奉献的点心配对分>=3点，则额外获得一枚茶券。
* **奖励：** 玩家获得老太君回赠的一个**更高等级**的随机空食器（从奖励牌堆抽取 L2 或 L3）。
    *   L1 -> 获得 L2
    *   L2 -> 获得 L3
    *   L3 -> 获得两个茶券

**D. 调整**

* 在你的回合，你可以任意调整等待区点心的位置（放在任意的盘子上），不消耗AP。

## 三、 老太君判定逻辑


玩家将制作好的点心献给老太君，虽不能品尝其美味（不计配对分），但能换取更高等级的食器，并积累“资历”。

### 1. 奉献规则
*   **动作：** 消耗 **1 AP**，将等待区的一组“食器+点心”移至奉献区。
*   **门槛（呈样）：** 被奉献的组合必须至少包含 **2点** 有效配对分。
*   **代价：** 奉献区的所有卡牌**不计算**“颜色/形状/材质”的配对得分。

### 2. 奉献奖励
当你进行奉献时，根据你献出的食器等级，老太君会回赠礼品（从奖励堆或拿取）：
*   献 **Level 1** 食器 $\rightarrow$ 获得 **Level 2** 食器（放入等待区）
*   献 **Level 2** 食器 $\rightarrow$ 获得 **Level 3** 食器（放入等待区）
*   献 **Level 3** 食器 $\rightarrow$ 获得 **2 枚茶券**

### 4. 获取玉盏：敬茶 (3 AP)

玩家必须同时满足以下条件，才能发起"敬茶"：
1.  **行动点消耗：** 消耗 **3 AP**。
2.  **资源消耗：** 消耗 **9个** 茶券。
    *   *叙事：需用平时积攒（品鉴）下来的上好茶叶，方能表敬意。*
3.  **资历折扣：** 奉献区每拥有 **1个** 盘子,则茶券消耗减少 **1个**。
    *   *叙事：只有多次奉献、资历够深的晚辈，才有资格向老太君敬茶。*

**效果：** 玉盏只作为标识物，不占用等待区/个人区/奉献区。持有玉盏者在最终结算时，奉献区分数翻倍。

---
## 四、 最终结算 (Final Scoring)

当满足以下条件时，游戏在进行到下一轮（公平轮）之后结束：
* **所有 L2 等级** 的食器都已被分发完毕。

### 1. 分值构成公式
每位玩家的最终得分 ($S$) 由以下各项加总构成：

$$S = \sum P_{ind} + C_{off} + T_{remain} - C_{wait} \times 2$$

如果持有玉盏，则 $C_{off} \times 2$

### 2. 各项变量详解

| 变量符号         | 名称             | 定义与计算方式                                                                       |
| :--------------- | :--------------- | :----------------------------------------------------------------------------------- |
| **$P_{ind}$**    | **个人区配对分** | 结算至个人区域的点心的配对分。                                                       |
| **$C_{off}$**    | **奉献区数量**   | 玩家献给老太君的点心组合的数量。持有玉盏则翻倍。                                     |
| **$T_{remain}$** | **剩余茶券**     | 游戏结束时，玩家剩余的茶券数量。                                                     |
| **$C_{wait}$**   | **滞留惩罚**     | 游戏结束时，留在玩家等待区（未结算）的上面存在点心的**盘子数量**。每盘扣除 **2分**。 |

### 3. 胜负判定
* **总分最高者获胜。**
* **平局判定：** 若总分相同，则奉献区盘数（$C_{off}$）较多的玩家获胜；若仍相同，则个人区内盘子较少的玩家获胜。

# 卡牌设计

### 一、 核心公式：维度矩阵

为了让配对既有挑战又有运气，我们采用 **3 x 3 x 2** 的非对称矩阵设计。

*   **视觉（颜色 C）：** 3种 —— 桃红、柳绿、鹅黄
*   **哲学（形状 S）：** 3种 —— 圆满（圆）、方正（方）、花韵（花）
*   **体感（材质 M）：** 2种 —— 暖（温润/陶木）、冷（清冽/琉璃）
    *   *设计意图：材质只有2种，意味着“奉献”的门槛较低（50%概率匹配），保证了游戏推进的速度（N值增长），符合“后盘必出”的节奏。*

---

### 二、 点心牌设计 (Dessert Deck)

点心是消耗品，也是分数的载体。

#### 1. 数量公式
$$Total = N_{basic} + N_{special} + N_{Epic} = 54 \text{ Cards}$$

*   **基础点心（Basic）：** 遵循标准分布。
    *   $$3(C) \times 3(S) \times 2(M) \times 2(R) = \mathbf{36张}$$
*   **稀有珍馐（Special）：** 双维度的复合点心。
    *   **双色（Twin-Color）：** 6张（红绿、红黄、绿黄各2张）。
    *   **双形（Twin-Shape）：** 6张（方圆、圆花、方花各2张）。
    *   **共计：** $\mathbf{12张}$
*   **顶级点心（Epic）：** 双维度双复合点心。
    *   **双色（Twin-Color）双形（Twin-Shape）：** 2张。
    *   **三色（Twin-Color）：** 2张。
    *   **三形（Twin-Color）：** 2张。
    *   **共计：** $\mathbf{6张}$


#### 2. 分布详情

**A. 基础分布 (54张)**
保证了每种特定组合（如：桃红-圆-暖）在牌堆里有 **3张**，提供稳定的配对基础。

**B. 复合分布 (6张 - 稀有掉落)**
这些卡牌在某一列拥有**两个实心点**，能单卡产出更高分值，适配 L2/L3 盘子。
*   **双色系 (鸳鸯)：**
    *   **红+绿** / 任意形 / 任意温 x 1
    *   **红+黄** / 任意形 / 任意温 x 1
    *   **绿+黄** / 任意形 / 任意温 x 1
*   **双形系 (攒盒)：**
    *   任意色 / **圆+方** / 任意温 x 1
    *   任意色 / **圆+花** / 任意温 x 1
    *   任意色 / **方+花** / 任意温 x 1

**C. 统计总览**
*   **颜色分布：** 红(20)、绿(20)、黄(20) —— *包含复合属性，完美平衡*。
*   **材质分布：** 暖食(30)、冷食(30) —— *复合牌随机分配温/冷属性*。

#### 3. 牌面叙事包装（示例）

**【基础点心】**
*   **桃红-圆-暖**：玫瑰赤豆糕
*   **柳绿-方-冷**：翡翠薄荷冻
*   **鹅黄-花-暖**：金桂海棠酥

**【稀有珍馐】（复合牌）**
*   **[双色] 桃红+柳绿**：**鸳鸯双色卷**
    *   *叙事：红白相间，翠羽映霞，寓意成双成对。*
    *   *玩法：在双色盘上可得2分颜色分。*
*   **[双色] 桃红+鹅黄**：**金玉满堂糕**
    *   *叙事：南瓜与枣泥层叠，金红交织，富贵逼人。*
*   **[双形] 圆+方**：**乾坤方圆盒**
    *   *叙事：外圆内方，模拟古代攒盒造型，一盒尽享两种口感。*
    *   *玩法：在“圆+方”兼容的盘子上可得2分形状分。*
*   **[双形] 花+圆**：**花好月圆饼**
    *   *叙事：中为圆芯，外以此花瓣包裹，极尽工巧。*

### 三、 食器牌设计 (Plate Deck)

食器是得分倍率器，采用 **金字塔稀有度** 设计。
食器同样拥有 C/S/M 三个属性，但属性代表的是 **“兼容性”**。

**总数量：42张 + 玉盏（特殊）**

#### Level 1：普通盘 (Common)
*   **数量**：20张
*   **特征**：**单属性严格限制**。
*   **公式**：每个盘子只有一个固定的颜色、形状和材质。
    *   例如：[红 / 圆 / 暖]
#### Level 2：精品盘 (Rare)
*   **数量**：16张 (老太君回赠的主要池子)
*   **特征**：**双维度宽容**。
*   **设计逻辑**：在“颜色”或“形状”上，具备 **2/3** 的兼容性；或者在“材质”上兼容所有。
*   **牌面示例**：
    *   **双色盘**：[红或黄 / 圆 / 暖] —— 颜色更容易匹配了。
    *   **通用材质盘**：[绿 / 方 / 冷或暖] —— 只要颜色形状对，材质必得分（但这会导致奉献门槛变低，容易流失，需权衡）。
#### Level 3：珍宝盘 (Epic)
*   **数量**：6张 (稀有掉落)
*   **特征**：**单维度全通（万能）**或者双维度双复合的盘子。
*   **牌面示例**：
    *   **百花盘**：[红 / **任意形状** / 冷] 。
    *   **流光盘**：[**任意颜色** / 方 / 暖] 。
    *   **彩釉盘**：[**两种颜色** / **两种形状** / 暖] 。

#### 特殊：玉盏 (Legendary)
*   **数量**：全场只存在一个。
*   玉盏只作为标识物，不占用等待/个人/奉献区域。持有玉盏者奉献区分数翻倍计算。
